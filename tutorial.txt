whoami - показать имя пользователя
apt - пакетный менеджер (команда утилитов для установки програмного обеспечения)
apt install - установить программу
apt remove - удалить программу
apt purge - удалить программу вместе со всеми зависимостями
apt list - показать список всех установленных программ
apt update - искать обновления по всем указанным репозиториям
git --version - посмотреть версию гита
systemctl - операция для служб
list-units - посмотреть на запущенные службы
.git директория - это место, где Git хранит метаданные и базу объектов проекта. Точка в начале обозраначет, что это скрытая директория (папка)
Область подготовленных файлов (staging area) - это файл, обычно располагающийся в .git директории, где содержится информация о том, какие изменения попадут в следующий коммит
Git имеет три уровня настроек:
local - настройки отдельного проекта
global - настройки уровня учетной записи
system - настройки операционной системы
Настройка глобальной системы:
git config --global user.name "Your name" - настройка имени
git config --global user.email youremail@gmail.com - настройка email
git config --global core.editore kate - настройка редактора (kate - редактор в Linux)
git config --global init.defaultBranch main - настройка ветки main
git config --global core.autocrif true - для организации переноса строк в Windows
git config --global core.autocrif input - для организации переноса строк в Linux
git config --list --show-origin - посмотреть установленные настройки (--show-origin показывает где они сохранены)
